# Samba de Delete (サンバ・デ・デリート)
悩みをサンバのリズムに乗せて吹き飛ばす、メンタルケア・ジョークツールです。

## 🚀 概要
「深刻な悩みも、陽気なリズムと演出があれば笑い飛ばせるはず」という仮説のもと開発しました。
入力したテキストが音楽に合わせてダイナミックに動き、最終的に画面外へ爆発・消失するUXを提供します。

## 🛠 使用技術
- **Frontend:** HTML5, CSS3 (Keyframes Animation), JavaScript
- **Design:** レスポンシブデザイン対応

## ✨ こだわったポイント
- **DOM操作とアニメーションの同期:** オーディオの再生タイミングに合わせて、JavaScriptで要素のクラスを制御し、躍動感のある動きを実現しました。
- **直感的な操作感:** 「入力して消す」というシンプルなフローを重視し、現場の忙しい合間でも一瞬で気分転換ができるように設計しています。
